CFG BAY / RUNTIME CONFIGURATION

配置舱

设备名、端口、并发流数、块大小和保存位置会写入本地配置，后续新任务读取最新参数。

点击按钮可直接打开本机目录选择器，也支持继续手动输入路径。
APPLY RULES

设置生效说明

1

即时生效

设备名称、QUIC 端口、并发流数、块大小会应用到后续新发起的任务。

2

需要重启

Web 主机地址与 Web 端口会保存到本地，但当前进程需要重启后才切换监听地址。
